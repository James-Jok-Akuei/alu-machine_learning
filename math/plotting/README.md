# Plotting

This folder contains scripts for various plotting techniques, including line plots, scatter plots, bar charts, frequency plots, and PCA visualizations.

## Files
- 0-line.py: Line plot
- 1-scatter.py: Scatter plot
- 2-change_scale.py: Scale change plot
- 3-two.py: Two-variable plot
- 4-frequency.py: Frequency plot
- 5-all_in_one.py: Combined plots
- 6-bars.py: Bar chart
- 100-gradient.py: Gradient visualization
- 101-pca.py: PCA plot

## Usage
Use these scripts to visualize data and mathematical concepts.
