# Unsupervised Learning

This folder contains scripts and resources for unsupervised learning techniques, including clustering, autoencoders, dimensionality reduction, and more.

## Subfolders
- autoencoders: Autoencoder models
- clustering: Clustering algorithms
- dimensionality_reduction: Dimensionality reduction methods
- hmm: Hidden Markov Models
- hyperparameter_tuning: Hyperparameter tuning scripts

## Usage
Explore each subfolder for specific unsupervised learning methods and examples.
