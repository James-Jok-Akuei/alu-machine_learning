# Clustering

This folder contains scripts and resources for clustering algorithms as part of the unsupervised learning assignment. Ensure all required Python files for each task are present in this directory.

## Tasks
- 0-initialize.py: Initialize K-means centroids
- 1-kmeans.py: K-means clustering
- 2-variance.py: Intra-cluster variance
- 3-optimum.py: Optimum k (elbow method)
- 4-initialize.py: GMM initialization
- 5-pdf.py: Gaussian PDF
- 6-expectation.py: EM expectation step
- 7-maximization.py: EM maximization step
- 8-EM.py: Full EM algorithm
- 9-BIC.py: Bayesian Information Criterion
- 10-kmeans.py: K-means with sklearn
- 11-gmm.py: GMM with sklearn
- 12-agglomerative.py: Agglomerative clustering

## Usage
Each script is standalone and can be run or imported as needed for the assignment.
